HTML

  • The head element contains the deep metadata that helps structure and run the webpgage.
  • Code in the head is usually very important and helps define the set up and elements of the rest
  • The body is used for content that is to be presented to the user.

  • Both head and body use Elements and Child Elements

CSS

  • There are three types of of styles of CSS, inline, internal and external.
  • A "margin" indicates how much space we want around the outside of an element.
  • A "padding" indicates how much space we want around the content insie an element.
  • The CSS "rule" is the structure of a line of CSS code, it includes a slector, then a declaration.
  • A slector defines the element of the HTML file you want to style
  • A declaration is a pair of key values which includes a CSS property and its value, it physical style you want to inplant.
